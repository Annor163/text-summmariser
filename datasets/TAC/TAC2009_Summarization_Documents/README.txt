		TAC 2009 Update Summarization and AESOP Documents


Created: March 1, 2011


1. Overview

This package contains test documents from the TAC 2009 Update
Summarization task.  These same documents are optional input in the
TAC 2009 AESOP task.

For task guidelines and additional information, please see the TAC
2009 Summarization Track Web page:
  http://www.nist.gov/tac/2009/Summarization/
