		TAC 2008 Update Summarization Documents


Created: March 1, 2011


1. Overview

This package contains test documents from the TAC 2008 Update
Summarization task.  Also included are additional annotations of TAC
2008 Update Summarization data that were contributed by participants
after the official evaluation.

For task guidelines and additional information, please see the TAC
2008 Summarization Track Web page:
  http://www.nist.gov/tac/2008/summarization/


2. Contents

UpdateSumm08_test_docs_files.tar.gz:
  Test document sets

2008 SCU-marked update corpus.zip:
  SCU-marked test documents 
(courtesy of University of Ottawa; 
see http://www.nist.gov/tac/publications/2008/participant.papers/UofO.proceedings.pdf)
