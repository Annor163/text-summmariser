		TAC 2010 Guided Summarization and AESOP Documents


Created: March 1, 2011

Revised: May 3, 2011
Added TAC 2010 Guided Summarization Sample document sets

1. Overview

This package contains sample and test documents from the TAC 2010
Guided Summarization task.  The test documents are optional input in
the TAC 2010 AESOP task.

For task guidelines and additional information, please see the TAC
2010 Summarization Track Web page:
  http://www.nist.gov/tac/2010/Summarization/
